# TEI_Transkription_Renner
TEI-XML Transkription eines handschriftlichen Entwurfs von Karl Renner („Entwurf einer provisorischen Verfassung“, 1918), inklusive Auszeichnung von Korrekturen (Streichungen, Ergänzungen, Ersetzungen) und strukturellen Einheiten. Erstellt im Rahmen der Übung "Aus dem Archiv ins Frontend. Digitale Editionen als statische Webseiten".
